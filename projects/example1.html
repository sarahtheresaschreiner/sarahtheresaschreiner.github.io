Project Details

Formula 1 Data Analysis

Completed: 20/07/2025

This project focuses on analyzing Formula 1 lap time data using machine learning models. The goal was to predict lap performance under different conditions and extract insights into race strategies.

Technologies Used

  • Python
  • Pandas
  • Scikit-learn
  • Matplotlib

Key Features

  • Data cleaning and preprocessing of lap time datasets
  • Exploratory data analysis with visualizations
  • Machine learning models for lap time prediction
  • Insights on driver consistency and tire strategies
