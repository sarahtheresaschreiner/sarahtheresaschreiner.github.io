    Data Visualitzation (With matplotlib and Seaborn)

    Recommended Setup

    import pandas as pd
pd.plotting.register_matplotlib_converters()
import matplotlib.pyplot as plt
%matplotlib inline
import seaborn as sns
print("Setup Complete")

    Kinds of Plots

    • Lineplot: sns.lineplot(data=df, label='Despacito') or df[col]
    • Barchart: sns.barplot(x=flight_data.index, y=flight_data['NK'])
    • Heatmap: sns.heatmap(data=flight_data, annot=True) -> (annot=True to see the values)
    • Scatter plot: sns.scatterplot(x=df.x, y=df.y, hue=df.z) -> hue is optional to color based on that value
    • Regression line: sns.regplot(x=df.x, y=df.y)
    • Regression lines for all hue options: sns.lmplot(x="a", y="b", hue="c", data=df)
    • categorical scatter plot: sns.swarmplot(x=df.a, y=df.b)
    • Histogram (amount): sns.histplot(df['col'])
      • with hue (color coded): sns.histplot(data=df, x='col1', hue='col2')
    • Density (kernel density estimate (KDE)): sns.kdeplot(data=df['col'], shade=True)
      • with hue (color coded): sns.kdeplot(data=df, x='col1', hue='col2')
    • 2D KDE: sns.jointplot(x=df['col'], y=df['col2'], kind="kde")

    Styling

    • Width and Height: plt.figure(figsize=(14,6))
    • Title: plt.title("Daily Global Streams of Popular Songs in 2017-2018")
    • X-Label: plt.xlabel('Date')
    • Y-Label: plt.ylabel("Arrial delay (in minutes)")
    • Seaborn styling:
      • Themes: sns.set_style('dark')
        • "darkgrid", "whitegrid", "dark", "white" (default), "ticks"

    Which plot to use

    • Trends: Lineplot
    • Relationship:
      • Comparing Quantities: Barplot
      • Find paatterns in numbers: Heatmap
      • Relation between two continuous variables: Scatterplot
      • See linear relationship: Regression line (regplot or lmplot (for two colors))
      • Relationship between continuous and categorical variable: Categorical scatter
    • Distribution:
      • Single numerical: Histogram
      • single numerical estimated smooth distribution: KDE
      • corresponding KDE plots for each individual variable: 2D KDE
