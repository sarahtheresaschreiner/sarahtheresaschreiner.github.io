    Merging and joining datasets

    • Combine datasets into one df
    • concat([df1,df2])
    • e.g. two dfs with same cols can be added together
    • join()
    • left.join(right, lsuffix='_CAN', rsuffix='_UK')
    • if in right is a matching index it will be added to left, the col names get a suffix
    • pd.merge(left,right, on='user_id')
    • join two dfs on a declared column, that is the same in both dataFrames.

    Handling Different Data Formats

    • Data comes in many formats: CSV, JSON, SQL, Excel, etc.
    • Pandas provides easy methods to read and write them

    CSV

    • Read CSV:
    df = pd.read_csv('file.csv')
    • Handle encoding issues:
    df = pd.read_csv('file.csv', encoding='utf-8')
df = pd.read_csv('file.csv', encoding='Windows-1252')
    • Write CSV:
    df.to_csv('output.csv', index=False)

    JSON

    • Read JSON:
    df = pd.read_json('file.json')
    • Write JSON:
    df.to_json('output.json', orient='records')
    • orient options: 'records', 'columns', 'index' depending on structure

    SQL

    • Requires a database connection:
    import sqlite3

conn = sqlite3.connect('database.db')
df = pd.read_sql('SELECT * FROM table_name', conn)
    • Write to SQL:
    df.to_sql('table_name', conn, if_exists='replace', index=False)
    • Works with SQLite, MySQL, PostgreSQL, etc.
